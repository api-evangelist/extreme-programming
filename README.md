# Extreme Programming (extreme-programming)
Extreme Programming (XP) is an agile software development methodology that aims to produce higher quality software and improve quality of life for the development team. XP emphasizes customer satisfaction through frequent releases, continuous integration, test-driven development, pair programming, and collective code ownership.
